# 🎨 Handycraft - Authentic Handmade Products

Welcome to **Handycraft**, a premium e-commerce platform designed for those who appreciate the beauty of hand-wrought artistry. From exquisite jewelry to rustic home decor, we bring the world's finest craftsmanship to your fingertips. ✨

---

## 🚀 Key Features

*   **🛒 Seamless Shopping:** A smooth, intuitive cart and checkout experience.
*   **💎 Curated Categories:** Easily browse through Jewelry, Home Decor, Apparel, and more.
*   **🔐 Secure Auth:** Robust registration and login system for a personalized experience.
*   **📱 Fully Responsive:** Optimized for everything from giant monitors to small smartphones.
*   **✨ Premium UI:** Elegant animations and a modern, artisan-inspired design.
*   **⚡ High Performance:** Lightweight vanilla JS and optimized PHP backend.

---

## 🛠️ Tech Stack

### Frontend
*   **HTML5 & CSS3** - Semantic structure and modern layouts.
*   **Vanilla JavaScript** - Fast, framework-free client-side logic.
*   **Font Awesome** - Beautiful iconography.
*   **Custom Animations** - Smooth CSS transitions for a premium feel.

### Backend
*   **PHP** - Reliable server-side processing and API management.
*   **MySQL** - Relational database for products, users, and orders.
*   **PDO** - Secure database interactions with prepared statements.

```text

│   └── 📄 *.sql          # Database schema migrations

```

---

## ⚙️ Development Setup

1.  **Database:** Import the `backend/handycraft.sql` file into your MySQL server.
2.  **Config:** Update `backend/config/Database.php` with your local database credentials.
3.  **Server:** Host the project on a local PHP server (like XAMPP, WAMP, or Laravel Herd).
4.  **Launch:** Open `frontend/index.html` in your browser!
